## Documentação da API

#### Retorna todos os itens

```http
  GET /api/items
```

| Parâmetro   | Tipo       | Obrigatório |Descrição                           |
| :---------- | :--------- | :---------  |:---------------------------------- |
| `Id` | `Guid` | **Sim** | O ID da Pessoa que você quer |

#### Retorna uma Pessoa

```http
  GET /api/items/${id}
```

| Parâmetro   | Tipo       | Obrigatório |Descrição                           |
| :---------- | :--------- | :---------  |:---------------------------------- |
| `id`      | `string` | **Sim** | O ID do Departamento que você quer |

#### Retorna um Departamento

Recebe dois números e retorna a sua soma.
